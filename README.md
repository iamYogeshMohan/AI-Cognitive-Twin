## 🚀 Key Features

### 🎙️ Hybrid Neural Voice Studio
*   **Zero-Cost Local Cloning**: Powered by **XTTS v2**, the system clones your exact voice locally without expensive cloud subscriptions.
*   **Edge-Neural Integration**: High-fidelity, real-time synthesis for regional Indian languages including **Tamil, Telugu, and Kannada**.
*   **Automatic Handshake**: Intelligent switching between local cloning and premium cloud voices (ElevenLabs).

### 🌍 Universal Linguistic Core
*   **Multi-Directional Resonance**: Advanced support for English, Hindi, Tamil, Telugu, Kannada, Spanish, and French.
*   **Localized UI**: Every corner of the workspace, from sidebars to footers, adapts to your selected linguistic resonance.

### 🧩 Deep Psychometric Onboarding
*   **8-Phase Initialization**: Maps your logic centers, biography, problem-solving traits, and "North Star" values.
*   **Persona Diversity**: Create different twins—**Personal**, **Legacy** (Memorial), or **Expertise** modes.

### 🛡️ Ethical Guardrails & Sovereignty
*   **Gemini-Style Disclosure**: Persistent, low-profile footers clarify the twin's role as a digital reflection, ensuring ethical clarity.
*   **Local-First Privacy**: Your identity, samples, and API keys remain on your hardware.

## 🍱 Installation & Setup

### 1. Clone the Workspace
```bash
git clone https://github.com/iamYogeshMohan/AI-Cognitive-Twin.git
cd AI-Cognitive-Twin
```

### 2. Neural Voice Server Setup
Ensure you have Python 3.10+ installed.
```bash
pip install TTS flask flask-cors torch torchaudio edge-tts
python voice_server.py
```

### 3. Frontend Initialization
```bash
npm install
npm run dev
```
